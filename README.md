# ☕ Coffee Day Cafe — Landing Page | Oasis Infobyte Task 1

A responsive coffee shop landing page built using **HTML** and **CSS**, developed as part of the **Oasis Infobyte Web Development Internship** (Task 1).

---

## 📌 Project Overview

This is a static landing page for **Coffee Day Café**, a fictional coffee shop based in Mumbai. The page showcases the café's brand, featured coffee products with prices, an about section, and contact details — designed to give visitors a warm, inviting first impression.

---

## ✨ Features

- **Navbar** — Logo with coffee icon, navigation links (About Us, Our Product, Delivery), and a search bar
- **Hero Section** — Welcome banner with the café name and a tagline
- **Featured Coffee** — Highlighted product (Americano) with image, price, and an Order Now button
- **Coffee Menu** — Grid of 4 coffee varieties with images, prices, and Add to Cart buttons:
  - Cappuccino (₹150)
  - Double Espresso (₹220)
  - Latte (₹240)
  - Mint Chocolate Mocha (₹250)
- **View More** — Button to expand the product listing
- **About Section** — Café owner profile with photo and café description
- **Footer** — Social media icons (Instagram, Facebook, Twitter, LinkedIn) and contact info (phone & email)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/SamruddhiNadgouda/Oasis-Infobyte-Task1-LandingPage-CoffeeDayCafe.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Oasis-Infobyte-Task1-LandingPage-CoffeeDayCafe
   ```

3. **Extract images**
   - Unzip `landingpage-images.zip` into the same folder so all images load correctly.

4. **Open in browser**
   - Open `landing page.html` in any modern web browser.

> No build tools or dependencies required — it's a pure HTML/CSS project!
